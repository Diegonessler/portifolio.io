Sobre o Projeto

Este projeto tem como objetivo aprender um pouco mais sobre criação de paginas WEB, utilizando as tecnologias mais modernas para garantir desempenho e escalabilidade. Durante o desenvolvimento, focamos em questão de responsividade e questão de banco de dados, tendo a difculdade de conectar com o banco em certos momentos.

Tecnologias Utilizadas

  • Frontend: HTML
  • Backend: PHP
  • Banco de Dados: MSQL
  • Outras: XAMPP
